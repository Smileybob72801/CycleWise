# Conventional Commits

[![Conventional Commits](https://img.shields.io/badge/Conventional%20Commits-1.0.0-%23FE5196?logo=conventionalcommits&logoColor=white)](https://conventionalcommits.org)

CycleWise follows the [**Conventional Commits 1.0.0**](https://github.com/conventional-commits/conventionalcommits.org)
specification for all commit messages.

This standard defines a simple, human- and machine-readable format that:

- keeps commit history consistent and meaningful,
- enables automated changelog generation, and
- aligns directly with Semantic Versioning (SemVer).

**TL;DR**

<type>[optional scope][!]: <short description>

[optional body]

[optional footer(s)]

Examples:
```text
feat(shared): add hydration insights
fix(ui): correct layout for period summary
chore(deps): update Kotlin to 2.0.0

Refer to the official documentation for full details and examples:
