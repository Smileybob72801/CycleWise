# Screenshot Plan

Play Store screenshot set and feature graphic design brief for RhythmWise.

---

## Device Frame

- **Device:** Pixel 8
- **Resolution:** 1080 x 2400 px
- **Orientation:** Portrait
- **Frame style:** Device art with shadow (use Android Studio device frames or
  screenshots.pro)

---

## Data Source

Use the **tutorial seed data** (`TutorialSeederUseCase`) for all screenshots. Do not
capture screenshots using real health data.

---

## Screenshots (8 total)

Google Play allows up to 8 screenshots per device type. The screenshots below are ordered
for maximum store impact — the first three are visible before the user taps "See more."

| # | Screen | Description | Key Elements to Highlight |
|---|--------|-------------|---------------------------|
| 1 | Passphrase lock screen | Security-first first impression | Lock icon, passphrase field, tagline about local encryption |
| 2 | Daily Log — Wellness tab | Core daily logging experience | Mood, energy, and libido selectors; water intake ring |
| 3 | Daily Log — Period tab | Period-specific logging | Flow intensity picker, color selector, consistency selector |
| 4 | Tracker — Calendar view | Cycle history at a glance | Calendar with color-coded cycle phases, day detail sheet |
| 5 | Insights screen | Cycle intelligence | Cycle length chart, next period prediction, fertile window estimate |
| 6 | Settings screen | Privacy controls | Data deletion option, reminder configuration, autolock toggle |
| 7 | Daily Log — Dark mode | Dark theme variant | Same Daily Log Wellness tab in dark mode to show theme support |
| 8 | Coach mark walkthrough | Onboarding experience | Active tooltip overlay on a screen to show guided first-run |

## Feature Graphic

The feature graphic is displayed at the top of the Play Store listing page.

- **Dimensions:** 1024 x 500 px
- **Format:** PNG or JPEG, no alpha

### Layout

```
+------------------------------------------------------------------+
|                                                                    |
|   [App Icon]     RhythmWise                                       |
|                  Privacy-first cycle tracking                      |
|                                                                    |
+------------------------------------------------------------------+
```

### Design Specifications

- **Background:** Horizontal gradient from primary purple to dusty rose
  - Left: `#8E6C88` (primary purple)
  - Right: `#B5808E` (dusty rose)
- **App icon:** Centered vertically on the left third, 200 x 200 px
- **Title text:** "RhythmWise" in white, bold, 48 pt equivalent
- **Tagline text:** "Privacy-first cycle tracking" in `#FFF7F3` (soft cream), regular
  weight, 28 pt equivalent
- **Text alignment:** Left-aligned beside the icon, centered vertically
- **Safe zone:** Keep 80 px padding on all edges — Play Store crops may vary

### Brand Colors Reference

| Name | Hex | Usage |
|------|-----|-------|
| Primary purple | `#8E6C88` | Gradient start, primary brand color |
| Dusty rose | `#B5808E` | Gradient end, accent |
| Soft cream | `#FFF7F3` | Tagline text, light backgrounds |
| White | `#FFFFFF` | Title text |
